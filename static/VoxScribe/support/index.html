Before You Contact Support

A few checks resolve the most common issues with model downloads, audio import, and export behavior.

Model download problems

Confirm the Mac has network access if you select a model that is not already available locally. Try the base model for a smaller first download.

Unsupported audio files

Try exporting the recording as WAV, MP3, M4A, or another common audio format before importing it into VoxScribe.

Slow transcription

Use a smaller Whisper model for faster processing, especially on long recordings or Macs with limited memory.

Project exports

Use the VoxScribe project export when you need to preserve text, annotations, codes, themes, and the current workspace state.
